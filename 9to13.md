```java
@Service
public class DocServiceImpl extends ServiceImpl<DocMapper, Doc> implements IDocService {

    @Autowired
    private SnowFlake snowFlake;
    @Autowired
    private IContentService contentService;

    @Override
    public void save(DocSaveReq req) {
        Doc doc = CopyUtil.copy(req, Doc.class);
        Content content = CopyUtil.copy(req, Content.class);

        if (ObjectUtils.isEmpty(req.getId())) {
            long id = snowFlake.nextId();
            doc.setId(id);
            doc.setViewCount(0);
            doc.setVoteCount(0);
            this.baseMapper.insert(doc);

            content.setId(id);
            contentService.save(content);
        } else {
            this.baseMapper.updateById(doc);
            boolean b = contentService.updateById(content);
            if (!b) {
                contentService.save(content);
            }
        }
    }
}
```

#### 3.2.4 UserServiceImpl

```java
@Service
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements IUserService {

    @Override
    public UserLoginResp login(UserLoginReq req) {
        User user = this.baseMapper.selectByLoginName(req.getLoginName());
        if (user == null) {
            throw new BusinessException(BusinessExceptionCode.LOGIN_ERROR);
        }
        String encrypted = DigestUtils.md5DigestAsHex(req.getPassword().getBytes());
        if (!encrypted.equals(user.getPassword())) {
            throw new BusinessException(BusinessExceptionCode.LOGIN_ERROR);
        }
        return CopyUtil.copy(user, UserLoginResp.class);
    }

    @Override
    public void save(UserSaveReq req) {
        User user = CopyUtil.copy(req, User.class);
        if (ObjectUtils.isEmpty(req.getId())) {
            user.setPassword(DigestUtils.md5DigestAsHex(req.getPassword().getBytes()));
            this.baseMapper.insert(user);
        } else {
            user.setPassword(DigestUtils.md5DigestAsHex(req.getPassword().getBytes()));
            this.baseMapper.updateById(user);
        }
    }
}
```

#### 3.2.5 LoginInterceptor.java

```java
@Component
public class LoginInterceptor implements HandlerInterceptor {

    private static final Logger LOG = LoggerFactory.getLogger(LoginInterceptor.class);
    @Resource
    private RedisTemplate redisTemplate;

    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) throws Exception {
        if (request.getMethod().toUpperCase().equals("OPTIONS")) return true;

        String uri = request.getRequestURI();
        // 白名单：不需要登录即可访问的接口
        if (uri.startsWith("/doc/vote/") || uri.startsWith("/doc/info/") ||
            uri.startsWith("/doc/all/") || uri.startsWith("/doc/list") ||
            uri.startsWith("/doc/save") || uri.startsWith("/doc/remove") ||
            uri.startsWith("/ebook/list") || uri.startsWith("/ebook/save") ||
            uri.startsWith("/ebook/update/") ||
            uri.startsWith("/category/all") || uri.startsWith("/category/save") ||
            uri.startsWith("/content/findContent/") ||
            uri.startsWith("/user/login") || uri.startsWith("/user/list") ||
            uri.startsWith("/user/save") || uri.startsWith("/user/resetPassword") ||
            uri.startsWith("/ebookSnapshot/") || uri.startsWith("/error")) {
            return true;
        }

        String token = request.getHeader("token");
        if (token == null || token.isEmpty()) {
            response.setStatus(401);
            return false;
        }
        Object object = redisTemplate.opsForValue().get(token);
        if (object == null) {
            response.setStatus(401);
            return false;
        }
        return true;
    }
}
```

## 4. 海洋生物文档阅读点赞功能

### 4.1 功能概述

阅读文档时自动增加阅读数，点赞限制同一 IP 每天只能点赞一次（通过 Redis 实现）。同时通过 WebSocket 推送实时更新。

### 4.2 后端实现

#### 4.2.1 DocServiceImpl.vote()

```java
@Override
public void vote(Long id) {
  String ip = RequestContext.getRemoteAddr();
  String key = "DOC_VOTE_" + id + "_" + ip;
  boolean flag = redisUtil.validateRepeat(key, 3600 * 24);
  if (flag) {
    this.baseMapper.increaseVoteCount(id);
  } else {
    throw new BusinessException(BusinessExceptionCode.VOTE_REPEAT);
  }
  // WebSocket 推送更新
  wsServiceAsync.updateInfo(id);
}
```

#### 4.2.2 RequestContext —— 线程变量存储 IP

```java
public class RequestContext implements Serializable {
    private static ThreadLocal<String> remoteAddr = new ThreadLocal<>();
    public static String getRemoteAddr() { return remoteAddr.get(); }
    public static void setRemoteAddr(String addr) { remoteAddr.set(addr); }
}
```

#### 4.2.3 LogAspect —— AOP 中获取 IP 并存入 RequestContext

```java
@Before("controllerPointcut()")
public void doBefore(JoinPoint joinPoint) throws Throwable {
    HttpServletRequest request = attributes.getRequest();
    // 获取用户远程 ip 并放入线程共享变量
    RequestContext.setRemoteAddr(getRemoteIp(request));
}

public String getRemoteIp(HttpServletRequest request) {
    String ip = request.getHeader("x-forwarded-for");
    if (ip == null || ip.length() == 0 || "unknown".equalsIgnoreCase(ip)) {
        ip = request.getHeader("Proxy-Client-IP");
    }
    if (ip == null || ip.length() == 0 || "unknown".equalsIgnoreCase(ip)) {
        ip = request.getRemoteAddr();
    }
    return ip;
}
```

#### 4.2.4 DocMapper.xml

#### 5.2.3 EbookSnapshotServiceImpl

**定时任务（每 30 秒测试/每 5 分钟正式）生成快照：**

```java
@Scheduled(fixedRate = 300000)  // 5分钟
public void genSnapshot() {
    LOG.info("生成今日快照开始");
    // 1. 查询所有电子书
    List<Ebook> ebookList = ebookMapper.selectList(null);
    // 2. 获取当前日期
    Date today = new Date();
    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
    String todayStr = sdf.format(today);
    // 3. 遍历电子书，更新快照
    for (Ebook ebook : ebookList) {
        this.baseMapper.genSnapshot(ebook.getId(), todayStr);
    }
    LOG.info("生成今日快照结束");
}
```

**EbookSnapshotMapper.xml：**
```xml
<update id="genSnapshot">
    insert into ebook_snapshot(ebook_id, `date`, view_count, vote_count, view_increase, vote_increase)
    select #{ebookId}, #{date},
           ifnull(e.view_count, 0), ifnull(e.vote_count, 0),
           ifnull(e.view_count, 0) - ifnull(pre.view_count, 0),
           ifnull(e.vote_count, 0) - ifnull(pre.vote_count, 0)
    from ebook e
    left join (
        select ebook_id, view_count, vote_count
        from ebook_snapshot
        where `date` = date_sub(#{date}, interval 1 day)
    ) pre on pre.ebook_id = e.id
    where e.id = #{ebookId}
    on duplicate key update
        view_count = ifnull(e.view_count, 0),
        vote_count = ifnull(e.vote_count, 0),
        view_increase = ifnull(e.view_count, 0) - ifnull(pre.view_count, 0),
        vote_increase = ifnull(e.vote_count, 0) - ifnull(pre.vote_count, 0)
</update>
```

## 6. 实战修复记录

### 6.1 CommonResp 构造函数为空
- **问题：** 构造函数 `public CommonResp(boolean b, String 查询成功, Object o) {}` 为空，参数未被赋值给字段
- **修复：** 使用 `@AllArgsConstructor` + `public CommonResp() {}`

### 6.2 PageReq 分页参数类型问题
- **问题：** `int page`、`int size` 默认值为 0，`@NotNull` 对基本类型无效。MyBatis-Plus 在 `size=0` 时只返回 count 不返回数据
- **修复：** 改为 `Integer page = 1`、`Integer size = 10`

### 6.3 页面刷新后菜单同步
- **问题：** 刷新页面后 Header 菜单和侧边栏高亮未同步到当前路由
- **修复：** 添加 `onMounted` + `watch` 监听 `route.path`，动态更新 `selectedKeys`

### 6.4 文档编辑报 Slate 节点错误
- **问题：** wangEditor 同时用了 `v-model` 和 `setHtml()`，内容冲突
- **修复：** 去掉 `v-model`，统一用 `setHtml()` / `getHtml()` 控制

### 6.5 点赞 SQL 写错
- **问题：** `increaseVoteCount` 的 SQL 中 `set vote_count = view_count + 1` 写成了 `view_count`
- **修复：** 改为 `set vote_count = vote_count + 1`

### 6.6 401 拦截器不生效
- **问题：** 多次修改 `LoginInterceptor` 和 `SpringMvcConfig` 后，某些路径仍被拦截
- **修复：** 统一在两个文件中同步维护白名单列表，确保 `mvn clean compile` 清除缓存

### 6.7 CORS 预检失败（访问 IP 变化时）
- **问题：** 从 `localhost` 改为 `192.168.x.x` 访问时，OPTIONS 预检请求被拦截
- **修复：** `CorsConfig` 改用 `CorsFilter` Bean 方式注册（Filter 级别高于 Interceptor）
